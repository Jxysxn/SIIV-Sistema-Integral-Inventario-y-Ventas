
INSTRUCCIONES DE RESTAURACIÓN – BASE DE DATOS


1. Abrir phpMyAdmin o cliente MySQL de preferencia.

2. Crear una base de datos vacía llamada: "sistemainventarioventas" con cotejamiento: utf8mb4_general_ci

3. Seleccionar la base creada y abrir la pestaña "Importar".

4. Cargar el archivo: SIIV_BD_Backup_2025.sql

5. Presionar "Continuar" y esperar confirmación de éxito.

6. Crear los usuarios MySQL del sistema ejecutando el archivo: SIIV_Usuarios_MySQL.sql

7. Abrir el proyecto Java y configurar correcta conexión en formulario Login.

8. Iniciar sesión con cualquiera de los usuarios creados:
   admin / vendedor / analista

9. Verificar que el inventario, compras y ventas carguen correctamente.
