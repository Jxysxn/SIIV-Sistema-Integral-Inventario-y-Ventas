Cómo restaurar la base de datos

Abrir phpMyAdmin.

Crear una base de datos vacía llamada 'sistemainventarioventas'.

Seleccionar la base de datos creada.

Ir a la pestaña "Importar".

Seleccionar el archivo "SIIV_BD_Backup_2025.sql".

Presionar "Continuar".

Verificar que las tablas, vistas, procedimientos, funciones y triggers se hayan creado correctamente.
